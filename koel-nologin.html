How to remove/bypass koel login page

Before we start I should mention that following this post you most likely grant anyone (who is anticipated enough) a "www-data" shell if you run koel on a self hosted linux server.

If that didn't scare you, you are as crazy as me and can continue. :)

First make sure you have a running koel setup as described in the official installation guide.
Now look for a file similar to this /var/www/koel/public/build/assets/LoginForm.*.js where * are random digits. And finally modify the javascript code in the file like so. 

...const d={email:"demo@koel.dev",password:"demo"};t("");const
      r=t(m()?d.email:""),...

==> 
...const d={email:"demo@koel.dev",password:"demo"};t("");b.login("YOUR_EMAIL","YOUR_PASSWORD");const
      r=t(m()?d.email:""),...

Finally replace YOUR_EMAIL, YOUR_PASSWORD with the corresponding login you are trying to achieve.
Now if you visit your koel page you just have to refresh the page once again and you are logged in without submitting any credentials.
